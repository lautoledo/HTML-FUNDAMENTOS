Listas HTML

As listas HTML são utilizadas para agrupar elementos relacionados.

Exemplo de Lista Não Ordenada

Uma lista HTML não ordenada é criada com a tag <ul>. Cada item da lista é definido com a tag <li>.

Exemplo:

      <ul>
        <li>Café</li>
        <li>Chá</li>
        <li>Leite</li>
        <li>Suco</li>
      </ul>

Confira:

  • Café
  • Chá
  • Leite
  • Suco

Exemplo de Lista Ordenada

Uma lista HTML ordenada é criada com a tag <ol>. Cada item da lista também é definido com a tag <li>.

Exemplo:

      <ol>
        <li>Café</li>
        <li>Chá</li>
        <li>Leite</li>
        <li>Suco</li>
      </ol>

Confira:

  1. Café
  2. Chá
  3. Leite
  4. Suco

Lista de Descrição HTML

Além das listas não ordenadas e ordenadas, o HTML suporta listas de descrição.

Uma lista de descrição é criada com a tag <dl>. Cada termo é definido com a tag <dt>, e cada descrição é definida com a tag <dd>.

Exemplo:

      <dl>
        <dt>Café</dt>
        <dd>- bebida quente e escura</dd>
        <dt>Leite</dt>
        <dd>- bebida branca e fria</dd>
      </dl>

Confira:

Café
- bebida quente e escura
Leite
- bebida branca e fria
