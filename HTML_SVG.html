Gráficos HTML SVG

SVG (gráficos vetoriais escaláveis) SVG define gráficos baseados em vetores em XML, que podem ser incorporados diretamente em páginas HTML.

Os gráficos SVG são escaláveis ​​e não perdem qualidade se forem ampliados ou redimensionados:

SVG

SVG é compatível com todos os principais navegadores.

O que é SVG?

  • SVG significa gráficos vetoriais escaláveis
  • SVG é usado para definir gráficos baseados em vetores para a Web
  • SVG define gráficos em formato XML
  • Cada elemento e atributo em arquivos SVG pode ser animado
  • SVG é uma recomendação do W3C
  • SVG integra-se com outros padrões, como CSS, DOM, XSL e JavaScript

O elemento <svg>

O elemento HTML <svg> é um contêiner para gráficos SVG.

SVG possui vários métodos para desenhar caminhos, retângulos, círculos, polígonos, texto e muito mais.

Círculo SVG

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" /> </svg>

Retângulo SVG

<svg width="400" height="120"> <rect x="10" y="10" width="200" height="100" stroke="red" stroke-width="6" fill="blue" /> </svg>

Retângulo SVG com opacidade e cantos arredondados

<svg width="400" height="180"> <rect x="50" y="20" rx="20" ry="20" width="150" height="150" style="fill:red;stroke:black;stroke-width:5;opacity:0.5" /> </svg>

Estrela SVG

<svg width="300" height="200"> <polygon points="100,10 40,198 190,78 10,78 160,198" style="fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;" /> </svg>

Elipse e texto gradiente SVG

<svg height="130" width="500"> <defs> <linearGradient id="grad1"> <stop offset="0%" stop-color="yellow" /> <stop offset="100%" stop-color="red" /> </linearGradient> </defs> <ellipse cx="100" cy="70" rx="85" ry="55" fill="url(#grad1)" /> <text fill="#ffffff" font-size="45" font-family="Verdana" x="50" y="86">SVG</text> Sorry, your browser does not support inline SVG. </svg>

Diferenças entre SVG e Canvas

SVG é uma linguagem para descrever gráficos 2D em XML, enquanto o Canvas desenha gráficos 2D dinamicamente (com JavaScript).

SVG Canvas
Resolution independent Resolution dependent
Support for event handlers No support for event handlers
Good text rendering capabilities Poor text rendering capabilities
Slow rendering if complex You can save the resulting image as .png or .jpg
Not suited for game applications Well suited for graphic-intensive games
