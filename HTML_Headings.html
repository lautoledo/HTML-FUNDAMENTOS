Cabeçalhos HTML

Os títulos HTML são definidos com as tags <h1> a <h6>.

<h1> define o título mais destacado. <h6> define o título menos destacado.

Exemplo

        <h1>Heading 1/h1>
        <h2>Heading 2/h2>
        <h3>Heading 3/h3>
        <h4>Heading 4/h4>
        <h5>Heading 5/h5>
        <h6>Heading 6/h6>

Os títulos são cruciais

Os mecanismos de busca usam os títulos para entender a estrutura e o conteúdo de suas páginas da web.

Os usuários geralmente percorrem uma página pelos títulos. É importante usar os títulos para mostrar a estrutura do documento.

<h1> deve ser usado para os títulos principais, seguidos por <h2>, depois por <h3> e assim por diante.

Nota: Use títulos HTML apenas para títulos. Não use títulos para tornar o texto GRANDE ou em negrito.

Títulos maiores

Cada título HTML tem um tamanho padrão. No entanto, você pode especificar o tamanho de qualquer título com o atributo style, usando a propriedade font-size do CSS:

Exemplo

    <h1 style="font-size:60px;">Heading 1/h1>
