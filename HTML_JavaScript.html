A tag HTML <script>

A tag HTML <script> é usada para definir um script do lado do cliente (JavaScript).

O <script> elemento contém instruções de script ou aponta para um arquivo de script externo por meio do src atributo.

Os usos comuns do JavaScript são manipulação de imagens, validação de formulários e alterações dinâmicas de conteúdo.

Para selecionar um elemento HTML, o JavaScript geralmente usa o document.getElementById() método.

Este exemplo de JavaScript escreve "Olá JavaScript!" em um elemento HTML com id="demo":

<script>
document.getElementById("demo").innerHTML = "Olá JavaScript!";
</script>

Um gostinho de JavaScript

Aqui estão alguns exemplos do que o JavaScript pode fazer:

JavaScript pode alterar o conteúdo:

document.getElementById("demo").innerHTML = "Olá JavaScript!";

JavaScript pode alterar estilos:

document.getElementById("demo").style.fontSize = "25px";
document.getElementById("demo").style.color = "red";
document.getElementById("demo").style.backgroundColor = "yellow";

JavaScript pode alterar atributos:

document.getElementById("image").src = "picture.gif";

A tag HTML <noscript>

A tag HTML <noscript> define um conteúdo alternativo a ser exibido aos usuários que possuem scripts desabilitados em seus navegadores ou possuem um navegador que não suporta scripts:

<script>
document.getElementById("demo").innerHTML = "Olá JavaScript!";
</script>
<noscript>Sorry, your browser does not support JavaScript!</noscript>
