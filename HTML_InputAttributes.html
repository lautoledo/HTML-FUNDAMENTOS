Atributos de Entrada HTML

O atributo de valor

O atributo input valueespecifica um valor inicial para um campo de entrada:




Atributo somente leitura

O atributo input readonlyespecifica que um campo de entrada é somente leitura. Um campo de entrada somente leitura não pode ser modificado (no entanto, um usuário pode tabulá-lo, destacá-lo e copiar o texto dele). O valor de um campo de entrada somente leitura será enviado ao enviar o formulário!

<form> <label for="fname">First name:</label><br> <input type="text" id="fname" name="fname" value="John" readonly><br> <label for="lname">Last name:</label><br> <input type="text" id="lname" name="lname" value="Doe"> </form>
