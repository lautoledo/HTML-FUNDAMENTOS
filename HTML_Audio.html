Áudio HTML

O elemento HTML <audio> é usado para reproduzir um arquivo de áudio em uma página da web.

O elemento HTML <audio>

Para reproduzir um arquivo de áudio em HTML, use o <audio> elemento:

<audio controls>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Confira:

Áudio HTML - Como funciona

O controls atributo adiciona controles de áudio, como reproduzir, pausar e volume.

O <source> elemento permite que você especifique arquivos de áudio alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

O texto entre as tags <audio> e </audio> só será exibido em navegadores que não suportam o <audio> elemento.

HTML <audio> Reprodução automática

Para iniciar um arquivo de áudio automaticamente, use o autoplay atributo:

<audio controls autoplay>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Confira:

Observação: os navegadores Chromium não permitem a reprodução automática na maioria dos casos. No entanto, a reprodução automática silenciada é sempre permitida.

Adicione muted depois de autoplay para permitir que seu arquivo de áudio comece a ser reproduzido automaticamente (mas sem som):

<audio controls autoplay muted>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Formatos de áudio HTML

Existem três formatos de áudio suportados: MP3, WAV e OGG. O suporte do navegador para os diferentes formatos é:

Formato de arquivo Tipo de mídia
MP3 audio/mpeg
OGG audio/ogg
WAV audio/wav

File Format Media Type

Áudio HTML - Métodos, Propriedades e Eventos

O HTML DOM define métodos, propriedades e eventos para o <audio> elemento.

Isso permite carregar, reproduzir e pausar áudios, bem como definir duração e volume.

Existem também eventos DOM que podem notificá-lo quando um áudio começa a ser reproduzido, é pausado, etc.

Para obter uma referência completa do DOM, acesse nossa Referência de DOM de áudio/vídeo HTML.
